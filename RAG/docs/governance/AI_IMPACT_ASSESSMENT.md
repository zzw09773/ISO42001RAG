# AI 系統影響評估（AI System Impact Assessment）

> ISO 42001 條文 6.1.4 / 8.4（AI 系統影響評估）+ A.5 證據。
> 狀態：v1.1.0（2026-07-02 編製）。
> 事實性欄位已依系統現況預填；**影響評定結論與簽核由審查負責人親自填寫**。

## 1. 系統與使用脈絡

| 欄位 | 內容 |
|---|---|
| 系統 | ISO 42001 RAG 法律文件查詢系統 v1.1.0 |
| 部署 | 內網（不對公網）、使用者為國軍法律承辦人 |
| 輸出性質 | 附條文引用的中文回答，明確定位為 AI 輔助資訊，**非法律建議** |
| 自動化程度 | 無自動決策——系統不做出對個人生效的決定，最終判斷由承辦人負責 |

## 2. 利害關係人識別

| 利害關係人 | 與系統的關係 | 潛在影響 |
|---|---|---|
| 法律承辦人（直接使用者） | 以系統輔助查詢條文 | 錯誤回答可能誤導承辦判斷（R1/R2/R9） |
| 案件當事人（間接） | 承辦人的法律處置對象 | 承辦人若未查證 AI 輸出，可能影響其權益 |
| 營運單位 | 系統擁有者 | 合規責任、稽核結果 |
| 個資當事人 | 查詢內容可能涉及 | 日誌含查詢原文（已以 0o640 + 內網控制） |
